# Current time.
Get current timestamp.


## Get timestamp
`_z.now();`

* **_z** to access UnderZ library.
* **now** method/action name.

> **Returns:** timestamp _Number_


## Examples
```html
<script>
_z.now(); // timestamp = number of milliseconds since 1970/01/01
</script>
```
